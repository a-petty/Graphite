# Service Mesh Rollout — Project Summary

## Project Overview

The Service Mesh Rollout project was initiated in November 2024 and is led by Ryan O'Brien. The core team includes David Kim, Derek Washington.


## Technical Architecture

The technical stack includes Istio, Kubernetes, Envoy, Prometheus. 
The team selected Istio after evaluating several alternatives, based on performance and developer experience. 
The team selected Kubernetes after evaluating several alternatives, based on performance and developer experience. 
The team selected Envoy after evaluating several alternatives, based on performance and developer experience. 



## Key Decisions

1. Adopted Istio as the primary framework

2. Implemented CI/CD pipeline using GitHub Actions for automated testing and deployment

3. Established weekly sync meetings between Ryan O'Brien and stakeholders
