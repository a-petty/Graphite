# Developer Portal — Project Summary

## Project Overview

The Developer Portal project was initiated in August 2024 and is led by Sophie Martin. The core team includes Maria Santos, Angela Rivera.


## Technical Architecture

The technical stack includes Next.js, Tailwind CSS, MDX, Swagger. 
Next.js integrates well with the existing infrastructure and reduces operational complexity. 
Tailwind CSS was chosen for its strong community support and proven track record in production. 
MDX was chosen for its strong community support and proven track record in production. 



## Key Decisions

1. Adopted Next.js as the primary framework

2. Implemented CI/CD pipeline using GitHub Actions for automated testing and deployment

3. Established weekly sync meetings between Sophie Martin and stakeholders


## Timeline

- August 2024: Project kickoff, initial architecture design

- Ongoing: Active development and iteration


## Status

The project is actively under development.
