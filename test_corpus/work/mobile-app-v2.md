# Mobile App V2 — Project Summary

## Project Overview

The Mobile App V2 project was initiated in September 2024 and is led by Carlos Mendez. The core team includes Marcus Johnson, Nina Kowalski, Angela Rivera.

Goal: Major mobile app update with offline support, push notifications, and redesigned navigation.

## Technical Architecture

The technical stack includes React Native, Firebase, TypeScript, Expo.

React Native was selected for code sharing with web frontend team.

Firebase was selected for push notifications and real-time database sync.

TypeScript was selected for shared types with web frontend.

Expo was selected for faster development iteration and OTA updates.

## Key Decisions

1. Chose Apollo Server over Hasura for GraphQL (July 2024)
1. Selected React over Vue for Dashboard Redesign (August 2024)

## Timeline

- September 2024: Project kickoff and initial architecture design
- Ongoing: Active development and iteration

## Status

The project is actively under development.
