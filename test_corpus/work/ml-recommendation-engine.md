# ML Recommendation Engine — Project Summary

## Project Overview

The ML Recommendation Engine project was initiated in October 2024 and is led by Emily Zhang. The core team includes James Liu, Priya Patel.

Goal: Build personalized recommendation system to increase user engagement and conversion.

## Technical Architecture

The technical stack includes PyTorch, MLflow, Airflow, FastAPI.

PyTorch was selected for flexible model architecture for recommendation algorithms.

MLflow was selected for experiment tracking and model versioning.

Airflow was selected for scheduled model retraining pipelines.

FastAPI was selected for low-latency serving endpoint for real-time predictions.

## Key Decisions

1. Adopted Snowflake as primary data warehouse (June 2024)
1. Chose Apollo Server over Hasura for GraphQL (July 2024)

## Timeline

- October 2024: Project kickoff and initial architecture design
- Ongoing: Active development and iteration

## Status

The project is actively under development.
